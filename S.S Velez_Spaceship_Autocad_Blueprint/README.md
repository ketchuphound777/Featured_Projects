S.S. Velez — AutoCAD 2D Spacecraft Blueprint
The S.S. Velez is a custom spacecraft concept designed entirely as a 2D engineering blueprint in AutoCAD. The project includes multiple orthographic views and detailed mechanical layout work, created using precise drafting techniques.

About This Project
The core of this project is a complete 2D blueprint showing the structure, engine layout, and geometry of the spacecraft.

I also recovered an older .sat ACIS file, but its contents are unknown and it cannot be opened. It was likely exported during experimentation or as part of an early idea to explore a 3D version, but no 3D model or screenshots remain.

What is preserved is the full 2D design process, including:

early blueprint drafts

viewport layouts

engine block planning

hull geometry development

final completed blueprint screenshots

These files document the entire creation of the S.S. Velez from concept to finished 2D design.
