Alien Character Animation Project
This folder documents my Blender project where I modeled, UV‑mapped, textured, and rigged an alien character. The final animation—where the alien held and threw a spear—was completed but the final .blend file and render were not preserved.

About This Project
I still have an older version of the alien model from before animation and before textures were applied. This version includes the full rig, but the character is not animated and does not have textures in the file.

Alongside the older file, I preserved a full set of screenshots showing the development process, including:

early mesh versions

UV unwrapping

digital texture work (computer‑painted, not hand‑painted)

rigging and weight painting

posing and movement tests

I do not have screenshots of the spear, but the workflow leading up to that stage is fully documented.
