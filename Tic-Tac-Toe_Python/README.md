Tic‑Tac‑Toe — Python Terminal Game
This project is a fully playable Tic‑Tac‑Toe game built in Python. It runs in the terminal and allows two players to take turns placing X’s and O’s on a 3×3 board. The game includes input validation, win detection, tie detection, and a clean game loop structure.

Project Overview
The game uses a simple list‑based board representation and modular functions to handle:

displaying the board

player turns

win condition checks

tie detection

player switching

The program continues until a player wins or the board is full.

Features
Two‑player local gameplay

Input validation (prevents invalid or occupied positions)

Row, column, and diagonal win detection

Tie detection when the board is full

Simple, readable code structure

How It Works
The board is stored as a list of 9 positions

Players choose positions 1–9

The game checks for:

row wins

column wins

diagonal wins

ties

The active player flips between X and O each turn
